## 잔여 리스크 / 후속 과제

- manual real-worker smoke는 외부 인증/환경 차이 때문에 완전 자동화는 어렵지만, operator preflight/runbook/report workflow와 실제 2026-04-12 실행 기록까지 확보했다.
- soak/fault-injection은 실행 시간이 길어 CI 기본 번들에 바로 넣기 어렵지만, weekly deep verification bundle(`ops:verify:deep:weekly`)로 운영 cadence를 고정했다.
- Bun 종료 지연은 현재 workaround로 실사용은 복구됐고, probe evidence 및 issue draft를 축적했지만 Bun 업그레이드 시 재확인은 계속 필요하다.
